# Prompt & Personality System

## Overview

Browse, customize, and create AI agent personalities with different tones and behaviors.

## User Flows

- Browse personalities in a card grid
- View personality details and usage stats
- Edit system prompts, temperature, and models
- Test-run personalities in a sandbox with sample input

## Data Shapes

**Entities:** Personality, ExampleOutput

## Visual Reference

See `PersonalityGrid.png` and `PersonalityEditor.png` for the target UI design.

## Components Provided

- `PersonalityGrid` — Grid view for browsing personalities
- `PersonalityCard` — Individual personality card
- `PersonalityEditor` — Full-screen editor and sandbox
